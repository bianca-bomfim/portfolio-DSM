# Portfolio Profissional - Bianca Bomfim

Este é o repositório do meu portfólio pessoal, desenvolvido para apresentar minha trajetória acadêmica na FATEC Itaquera e minha atuação profissional como Desenvolvedora Full Stack focada em Risk Tech.

## 🚀 Tecnologias Utilizadas

O projeto foi construído com foco em performance e design moderno (UX/UI):

- **HTML5** (Estrutura semântica)
- **CSS3** (Variáveis, Flexbox, Grid, Glassmorphism e Animações Fluidas)
- **JavaScript** (Animações de scroll e manipulação de ícones)
- **Lucide Icons** (Biblioteca de ícones vetoriais)
- **Google Fonts** (Fonte Inter para legibilidade)

## ✨ Destaques do Projeto

- **Design Responsivo:** Adaptado para proporcionar a melhor experiência em dispositivos móveis e desktops.
- **Animações de Scroll:** Implementação de efeitos de *fade-in* para uma navegação suave.
- **Hero Section:** Introdução dinâmica com animação de "blob" orgânico na imagem de perfil.
- **Arquitetura CSS:** Uso de variáveis customizadas para facilitar a manutenção de cores e temas.
- **Showcase de Projetos:** Grade organizada exibindo a evolução técnica através dos Projetos Integradores (PI) da FATEC.

## 📂 Estrutura de Arquivos

- `index.html`: Documento principal com a estrutura da Landing Page.
- `style.css`: Folha de estilo contendo toda a identidade visual e animações.
- `script.js`: Lógica para animações de entrada e carregamento de ícones.
- `img/`: Recursos visuais do projeto.

## 👩‍💻 Sobre a Autora

**Bianca Bomfim** - Desenvolvedora Full Stack com foco em sistemas de alta disponibilidade para o setor financeiro. Atualmente integrante do time de Risk Tech no Delta Global Bank, desenvolvendo soluções para mitigação de fraude e análise de risco.
