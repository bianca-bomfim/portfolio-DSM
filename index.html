Sobre Mim

Desenvolvedora Full Stack com foco em sistemas de alta disponibilidade para o setor financeiro. Graduada em Desenvolvimento de Software Multiplataforma pela FATEC Itaquera (agosto/2023- julho/2026), onde obtive reconhecimento acadêmico que resultou em uma bolsa de intercâmbio para certificação internacional na Espanha.

Atualmente, atuo no time de Risk Tech do Delta Global Bank (outubro/2025), desenvolvendo soluções para mitigação de fraude e análise de risco. Tenho experiência prática na construção de motores de decisão e monitoramento transacional de larga escala.

Principais frentes

  • Backend & Risk Engine: APIs em Java (Spring Boot) e scripts Python para análise de risco.
  • Data & Dashboards: Interfaces em React para monitoramento de fluxos PIX em tempo real.
  • Arquitetura: Microsserviços com foco em idempotência e resiliência.
  • Projetos de Escala: Apps Web/Mobile para alto tráfego (Prof. Rafael Procópio).

Stack Tecnológica

  • Backend: Java (Spring Boot), Python, Node.js.
  • Frontend: React, React Native, TypeScript.
  • Infra/DB: PostgreSQL, REST APIs, Docker, Git.
