Sistema de Reservas de Salas.

Solução robusta para gestão e agendamento de laboratórios acadêmicos e salas de aula.

Descrição do Projeto

Este sistema foi desenvolvido para resolver a complexidade do agendamento de espaços acadêmicos. A plataforma permite o controle rigoroso de disponibilidades, evitando conflitos de horários e centralizando a gestão de recursos como equipamentos e softwares disponíveis em cada sala.

Minha Participação (Backend)

Como desenvolvedora do time de Backend, fui responsável pela construção da arquitetura da API. Implementei o sistema de autenticação e o controle de permissões por níveis (RBAC), garantindo que apenas administradores pudessem gerenciar recursos críticos. Também atuei na modelagem do banco de dados MySQL e na documentação dos endpoints.

PHP 8 MySQL Composer API REST RBAC

Destaques Técnicos

  • Gestão de Perfis: Hierarquia entre usuários Comuns, Administradores e Supremo.
  • Segurança: Implementação de tokens para validação de sessões e permissões.
  • CRUD de Recursos: Gerenciamento completo de Salas, Equipamentos e Softwares.
  • Logs e Status: Sistema automático de status de reserva (pendente/confirmada).
